## 项目结构

```
project/
├── app/
│   ├── main.py                  # FastAPI 后端入口文件
│   ├── models/
│   │   ├── llm_model.py         # LLM 模型（微调的语言模型）
│   │   └── rag_model.py         # RAG 模型（结合检索和生成）
│   ├── retrieval/
│   │   └── query_handler.py     # 检索模块（QueryHandler，使用 FAISS 或其他技术）
│   └── api/
│       └── query.py             # FastAPI 路由和请求处理
├── frontend/
│   └── app.py                   # Streamlit 前端入口文件
└── requirements.txt             # 项目依赖

```
